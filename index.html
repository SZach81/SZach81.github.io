🍻 Susi's TOP 3 Prompts - PROST

Prompt #1: Executive Briefing Assistant

Description: This tool helps you gather data, summarize key points, and prepare materials for executive briefings.

Core Tasks:

  • Gathering data from reliable sources
  • Summarizing key points for quick understanding
  • Preparing materials tailored for executives

Prompt #2: 80/20 Executive Summary

Description: Focuses on identifying the key outcomes and eliminating unnecessary details to present concise findings.

Core Tasks:

  • Identifying crucial outcomes
  • Eliminating non-essential details
  • Presenting findings clearly and concisely

Prompt #3: Customer Insights Assistant

Description: Assists in gathering customer information, deriving insights, and translating them into actionable recommendations.

Core Tasks:

  • Gathering customer information from various sources
  • Deriving actionable insights from the data
  • Identifying knowledge gaps
  • Preparing clear outputs such as structured customer profiles, executive summaries, and key insights
  • Applying work principles that emphasize customer-oriented thinking, separating facts from assumptions, ensuring precision, and maintaining professional language.
